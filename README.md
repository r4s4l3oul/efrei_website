# efrei_website

Projet de développement web réalisé dans le cadre de ma formation en cycle ingénieur à l'EFREI.

## Présentation

Ce projet consiste en la réalisation d'un site web présentant le département d'informatique de l'EFREI.

L'objectif était de concevoir un site statique moderne permettant de présenter :

* Le département d'informatique
* Les formations proposées
* Le corps enseignant et la recherche
* Les projets réalisés
* Les informations de contact
* Les actualités et événements

## Démo en ligne

🔗 https://r4s4l3oul.github.io/efrei_website/

## Technologies utilisées

* HTML5
* CSS3
* JavaScript

## Fonctionnalités

* Interface responsive
* Barre de navigation multi-pages
* Carrousel d'images dynamique
* Mise en page moderne
* Présentation des formations et activités du département
* Liens vers les ressources officielles de l'EFREI
